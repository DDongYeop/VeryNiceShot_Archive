## **_게임 소개_**

"Very Nice Shot"은 한정된 횟수 안에 홀에 공을 넣는 간단한 게임입니다

| 내용      |      설명      |
| :-------- | :------------: |
| 제목      |   Very Nice Shot   |
| 게임 장르 | 재주얼, 퍼즐, 스포츠  |

---

<br>

## **게임 특징**

- ### 머리를 써야 클리어 할 수 있는 퍼즐 맵들
- ### 스테이지마다 새로운 기믹들
- ### 세상에 없던 새로운 느낌의 게임
